Ping Pong!

These Are the Rules

Ping Pong will count up to your number with the following exceptions:

  1. Numbers divisible by 3 will return "Ping"
  2. Numbers divisible by 5 will return "Pong"
  3. Numbers divisible by 15 will return "Ping-Pong"
